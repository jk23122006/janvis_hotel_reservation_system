# Janvi's Hotel Reservation System
This is a simple SQL database project to manage hotel guests, rooms and staff members, bookings and payments.


## Tools Used:-
1. SQL Tools (Extension)
2. XAMPP Control Panel
3. phpmyadmin local host
4. GitHub


## Project Files:-
- tables.sql: Created the tables
- data.sql: Inserted sample data in the following tables
- queries.sql: Used SQL queries to test the data  base
- ER Diagram
- screenshots of tables and query result


## Tables
- **Guests:** stores customers details like name, last name email and number.
- **Rooms:** stores room details like room number, type, price and status.
- **Staff:** stores staff members names and roles.
- **Bookings:** reservations are stored here
- **Payments:** stores payment details
